# FocusSprint

A smart productivity timer built with React and Firebase.

## Features

- Adaptive focus sessions
- Break mode
- Google Authentication
- Firestore sync
- Focus score
- Daily goals
- Streak tracking
- Weekly analytics
- Ambient sounds

## Tech Stack

- React
- Firebase Authentication
- Firestore
- Netlify

## Run locally

```bash
npm install
npm start
```
